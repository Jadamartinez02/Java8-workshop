# Dominando Java 8

# Ejercicio 1: Tu propia Interfaz Funcional
Objetivo: Crear una interfaz que realice una operación matemática personalizada.

# Ejercicio 2: Filtrado y Transformación con Streams
Objetivo: Dada una lista de nombres, filtrar los que empiecen con "A", convertirlos a mayúsculas y ordenarlos alfabéticamente.

# Ejercicio 3: Uso de Predicate y Consumer
Objetivo: Usar las interfaces funcionales predefinidas de Java (java.util.function).

# Ejercicio 4: Procesamiento de Objetos (Stream + Map)
Objetivo: Tienes una lista de objetos Producto. Debes obtener una lista de String que contenga solo los nombres de los productos que cuestan más de 500.

# Ejercicio 5: Reducción de Datos (reduce)
Objetivo: Calcular el valor total de una lista de números (sumatoria) utilizando programación funcional. El método reduce combina todos los elementos en un solo resultado.

# Ejercicio 6: Agrupamiento de Datos (GroupingBy)
Objetivo: Dada una lista de personas, agruparlas por su ciudad. Este es un uso avanzado de Collectors.
